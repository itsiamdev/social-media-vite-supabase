# Community Forum & Q&A Site

A modern social media website built with React, Vite, TypeScript, Supabase, and Tailwind CSS.

This project contains the code from the YouTube tutorial by [PedroTech](https://www.youtube.com/@PedroTechnologies): https://www.youtube.com/watch?v=_sSTzz13tVY

## 🛠️ Tech Stack

| Technology | Description |
|------------|-------------|
| React | UI Framework |
| Vite | Build Tool |
| TypeScript | Language |
| Supabase | Backend (Auth, Database, Storage) |
| Tailwind CSS | Styling |

## ✨ Features

- GitHub Authentication
- Post Creation with Image Uploads
- Voting System (Like/Dislike)
- Commenting System with Nested Replies
- Community/Categories Support
- Glassmorphism UI Design
- Real-time Updates

## 🚀 Getting Started

### Prerequisites

- Node.js
- npm
- Supabase Account

### Installation

```bash
# Clone the repository
git clone https://github.com/machadop1407/your-repo-name.git
cd your-repo-name

# Install dependencies
npm install
```

### Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
